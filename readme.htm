Markdown-Temp

什么是Markdown-Temp？

Markdown-Temp是一款在线Markdown编辑器，她不仅免费开源，而且简单适用。Markdown-Temp适用于编写临时文档与在线分享。

Markdown-Temp说明

  • 编辑器使用Editor.md开发
  • 后端使用Parsedown进行解析
  • Markdown-Temp使用PHP进行开发
  • 使用了BootStrap框架

功能介绍

  • 实时预览
  • 支持导出HTML、.md等格式
  • 生成唯一链接方便临时分享
  • 30秒自动保存

其它说明

  • Markdown-Temp定位与临时在线文档，每个月的10号/25号会定期对用户文档进行清理，最长可保存15天，因此打算长期保存的用户并不适用，所以建议导出HTML或.md格式。
  • Markdown-Temp不支持上传图片，若您需要插入图片请使用小z图床
  • 大部分核心代码均使用开源程序，小z博客只是在已有的轮子上进行组装
  • 由于本人不是专业程序开发，代码写得比较渣，还请各位看官将就一下。
  • 如果您有什么好的建议或想法，欢迎在Issues中提出。
